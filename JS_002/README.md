# JavaScript 시작하기

## 변수

- 데이터를 저장하는 기억장소, 기억장소의 이름
- 변수를 선언할때는 const, let 라는 키워드를 사용한다
- 변수를 선언하는 것은 데이터를 저장할 장소를 예약하는 것

```
const num1 = 100
let num2 = 100
```

- HTML tag 요소를 변수에 보관하기
- 변수를 통하여 pTag의 속성(property)에 접근할 수 있다

```
const pTag = document.querySelector("p")
pTag.innerText = "우리나라"
pTag.style.color = "blue"
```
